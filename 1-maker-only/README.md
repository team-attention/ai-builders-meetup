## 후기

> "참석률 100% AI 메이커 밋업"

오직 메이커만 모이는, 진짜들의 이야기를 들으러 모였습니다. 30명까지 들어갈 수 있는 장소라, 딱 30분을 초대했는데 한 분도 빠짐없이 다 오셨어요.

**박상민** - 커서 사용량 상위 1% 유저는 어떻게 개발하는지 들었습니다. FE → 풀스택이 된 이야기가 공감이 됐어요.

**배휘동** - 실제 AX 현장에서 겪은 좌충우돌. 모든 건 개선 가능하다!는 믿음을 바탕으로 조직을 10x 하는 시도를 하고 계셨어요.

**천재윤** - 당근에서 실제로 운영중인 에이전트를 엿볼 수 있었어요. 생각보다 내부 업무에 엄청 많이 적용돼있어서 깜짝 놀랐어요.

**조용래** - LLM 시대의 검색. 리서치 기반 발표가 이렇게 재밌을 수 있다니!

**강동주** - OpenAI는 노션을 어떻게 쓰는지! OpenAI가 Notion을 너무 사랑해서 먼저 API를 열어준 이야기.
