## 소개

Claude Code를 사랑하는 Max 유저를 위해 Team Attention이 Anthropic의 후원을 받아 준비하는 이벤트입니다.

Claude Code의 Skills, Subagents, Commands, Hooks등을 직접 만드는 해커톤입니다.
하지만 단순한 해커톤이 아닙니다. 스피커 세션을 즐기고, 같은 고민을 하는 분들과 네트워킹하세요.

**스킬톤이란?**
3시간 동안 자신만의 Claude Code 워크플로우를 만들고, 결과물을 공유한 뒤 우승팀을 선정합니다.
Anthropic 공식 지원 하에 진행되며, 참가자들에게는 Claude Credit이 제공됩니다.

---

## 이런 고민을 하고 계신가요?

- **스킬 설계 패턴**: 스킬에서 서브에이전트를 부르는 게 맞을까? 아니면 서브에이전트가 스킬을 사용하는 게 더 좋은 패턴일까?
- **메타 워크플로우**: 어제 비슷한 문제를 겪었는데, 오늘도 그 문제를 겪고 있다. 워크플로우를 자동으로 관리할 수는 없을까?
- **조직 차원의 Claude Code 활용**: 우리 팀의 토큰 사용량, 자주 사용하는 스킬, 사용하지 않아서 제거해야 하는 워크플로우는 뭘까?
- **개발 외 업무에서의 활용**: 메일, 슬랙, 노션 같은 툴은 물론, 슬라이드, 비디오 편집까지. 비개발자가 쉽게 적용할 수 있는 방법은 뭘까?

이런 고민을 함께 나누고, 직접 만들어보는 자리입니다.
